## Требования

- Go 1.24 или выше
- git
- Taskfile (опционально)

## Конфигурация

1. На основе `.env.example` создать файл `.env.local`

2. Установить зависимости

    ```bash
    go mod tidy
    ```

3. Запустить через Taskfile (при наличии)

    ```bash
    task run
    ```
    или вручную
    ```bash
    go run ./cmd/http_server/main.go -config=./configs/local.yml
    ```
